Setting Up Lump Sum Billing Lines for Manual Calculation

From Contract Billing Processing (G52), choose Contract Billing Line Details (P5202)

If you do not want the system to calculate the billing amounts, you can set up a lump sum billing line for manual calculation. You calculate the billing amounts and enter them after the invoice batch is created. You can add amounts to these billing lines during the invoicing process, but if you run revenue recognition as a separate process, you are not able to add amounts to the billing lines in a revenue batch..
